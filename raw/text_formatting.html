Text Formatting Guide

Overview

The server supports a flexible rich-text formatting system used in chat, player tags, and UI elements. Formats are triggered by the ampersand (&) character followed by specific tags.

Color Formatting

You can change the text color using RGB values, HEX codes, or predefined color names.

Tag Format Description
&c[r g b] &c[255 0 0] Sets color using Decimal RGB (0-255).
&c[hex] &c[FF0000] Sets color using HEX (RGB or RGBA).
&r[] or &w[] &r[] Resets color to default (White).

Predefined Named Colors

You can also use standard color names inside the color tag:

  • &red[]
  • &green[]
  • &blue[]
  • &yellow[]
  • &orange[]
  • &purple[]
Note
Predefined colors are case-insensitive in most contexts.

Special Elements

Images

To insert an icon or an image directly into the text line:

  • Format: &i[filename]
  • Example: &i[addons/server/(warning icon)] Warning!

Hyperlinks

Make parts of the text clickable.

  • Format: &hl[url,underlined]
  • Parameters: * - url: The destination address (e.g., https://google.com).
  • underlined: 1 to show an underline, 0 to hide it.
  • Example: &hl[https://google.com,1]CLICK HERE
Note
Any format after the hyperlink removes the hyperlink, so hyperlinks of different colors will be very difficult to make.

Practical Examples

Here is how you can combine different formats:

Source Code Resulting Effect
&c[255 0 0]Ser&c[FF0000FF]ver Red "Ser", then Red "ver" (via Hex).
&green[]Online: &w12/20 Green "Online: " followed by white numbers.
&hl[https://discord.gg/link,1]&i[discord.png] Join us! A clickable Discord icon and text. (you can use discord icon from game files
or upload your server content and player will see your icon after installing your content
Important
Formatting tags are processed sequentially. If you forget to use &r, the color will "bleed" into the rest of the text line.
