AngelScript

Introduction

AngelScript is a free, open source, flexible, and cross-platform scripting library designed for embedding in applications. Its goal is to provide an easy-to-use library that is powerful yet not burdened by rarely used features.

Development History

Development of AngelScript began in February 2003, with the first public release on March 28th, 2003, featuring only basic functionality. Since then, the library has seen frequent releases with continuous improvements and new features.

Key Features

  • Flexible cross-platform scripting for application extensibility
  • Easy integration for both application programmers and script writers
  • Direct calling of standard C functions and C++ methods with minimal glue code
  • No duplication of functionality between application and scripts
  • Object-oriented scripting language with familiar C/C++ syntax

Technical Overview

The AngelCode Scripting Library (commonly known as AngelScript) allows applications to extend their functionality through external scripts. Applications simply register functions, objects, and methods, enabling scripts to work directly with the application's existing codebase.
